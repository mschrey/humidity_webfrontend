# humidity_webfrontend

This is the Web Frontend, which can present the temperature and humidity data in a somewhat nice fashion. 

## Installation ##
Clone the repo and then untar the jpgraph-4.3.2.tar.gz archive using 
``tar -xzvf jpgraph-4.3.2.tar.gz``

Then create a symbolic link using
``ln -s ./jpgraph-4.3.2 jpgraph``
